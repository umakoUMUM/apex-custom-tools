# Apexカスタムリザルト集計ツール

Apex Legendsのカスタムマッチで、スクリーンショットから戦績（順位・チーム名・キル数）を読み取って、スプレッドシートに書き込むツールです。

## ⚠️ 大事な注意点
- スクリーンショットは必ず **1920×1080** の解像度で撮影してください。
- それ以外の解像度だと、文字を読み取る場所がズレて正しく動きません。

## 🛠️ 準備
1. ライブラリのインストール
   ターミナルで以下を実行してください。
   pip install -r requirements.txt

2. ファイルの配置
   - .env: スプレッドシートIDを記入
   - credentials.json: Google Cloudの認証キーを置く
   - config.yaml: チーム数（現在は17チーム設定）などを確認

## 📝 使い方
1. screenshots フォルダに画像を入れます。
2. プログラムを実行します。
   python main.py

3. 処理が終わると画像は自動的に processed フォルダに移動します。

## 🔍 テスト用ツール
- visual_check.py: 
  - 画像が正しく切り出せているか本番前に確認できます。
  - 実行すると debug_crops フォルダにテスト画像が保存されます。
