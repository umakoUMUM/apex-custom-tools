# Apexカスタムリザルト集計ツール

Apex Legendsのカスタムマッチで、スクリーンショットから戦績（順位・チーム名・キル数）を読み取って、スプレッドシートに書き込むツールです。

## ⚠️ 大事な注意点
- スクリーンショットは必ず **1920×1080** の解像度で撮影してください。
- それ以外の解像度だと、文字を読み取る場所がズレて正しく動きません。

## 📊 セルの色について
書き込み時、確認が必要な箇所には自動で色が付きます。

- **オレンジ色（補正）**:
  - 「09キル」が「60キル」と誤読されたのを、ツールが自動で「9キル」に直した箇所です。
  - 数値が合っているか確認してください。
- **赤色（異常）**:
  - 順位が0位だったり、20キルを超えていたり、読み取りミスが疑われる箇所です。
  - 画像を見て手動で修正してください。

## 🛠️ 準備
1. ライブラリのインストール
   pip install -r requirements.txt


2. ファイルの配置
   - .env: スプレッドシートIDを記入
   - credentials.json: Google Cloudの認証キーを置く
   - config.yaml: チーム数などを確認

## 📝 使い方
1. screenshots フォルダに画像を入れます。
2. プログラムを実行します。
   python main.py

3. 処理が終わると画像は自動的に processed フォルダに移動します。

## 🔍 テスト用ツール
- visual_check.py:
  - 画像が正しく切り出せているか確認できます。
  - 実行すると debug_crops フォルダにテスト画像が書き出されます。
